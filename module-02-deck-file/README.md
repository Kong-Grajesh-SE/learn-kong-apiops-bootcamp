# Module 02 - deck file Commands

> **Scenario.** You can sync config to Kong, but now you need to validate, lint, and transform config files offline - convert OpenAPI specs, merge multi-file setups, and patch values without touching the gateway.

## Module outcomes
- Validate and lint config offline with `deck file validate` and `deck file lint`
- Convert OpenAPI specs to Kong config with `deck file openapi2kong`
- Merge, render, and patch multi-file configurations
- Manage plugins and tags with `deck file add-plugins`
