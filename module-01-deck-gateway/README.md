# Module 01 - deck gateway Commands

> **Scenario.** You've built Kong config through the Admin API or Konnect UI. Now learn the `deck gateway` commands to capture, compare, and deploy configuration from the command line.

## Module outcomes
- Connect to Kong with `deck gateway ping`
- Snapshot live state with `deck gateway dump`
- Preview changes safely with `deck gateway diff`
- Deploy config with `deck gateway sync` and `deck gateway apply`
