# Module 03 - Multi-Team Workflows

> **Scenario.** Multiple teams share one Kong instance. You need to split config by team with tags, run full change workflows, drive config from OpenAPI specs, and practice backup and recovery.

## Module outcomes
- Split config by team using tags and `--select-tag`
- Run a full change workflow: edit, validate, lint, diff, sync
- Drive gateway config from OpenAPI specifications
- Practice backup and recovery procedures
